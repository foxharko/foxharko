# Українська - Харко / Harko
Harko / Харко — український антропоморфний лис-характерник із Запорозької Січі. Майстер омани й тиші, що маскує військо від ворога та знаходить слід там, де інші бачать лише степ. fox_harko 
# English - Харко / Harko
Harko / Харко is a Ukrainian anthropomorphic fox and Cossack-inspired furry OC from the Zaporizhian Sich. A silent master of illusion who hides troops from enemies and finds tracks where others see only steppe. fox_harko
# 日本語 - Harko / Харко / ハルコ
Harko / Харко / ハルコは、ウクライナのザポロージャ・シーチに着想を得た狐の獣人キャラクター。幻惑と静けさを操り、敵から仲間を隠し、草原に残るわずかな痕跡を見つける。fox_harko
# 简体中文 - Harko / Харко / 哈尔科
Harko / Харко / 哈尔科是受乌克兰扎波罗热哥萨克传说启发的狐狸兽人原创角色。他擅长幻术与潜行，能隐藏同伴，并在草原中发现别人看不见的踪迹。fox_harko
